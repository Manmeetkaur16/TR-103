* **Storing Register User Data Using `useState()`**

  * Created form fields for:
    * Name  
    * Email  
    * Password  
  * Used a single state object to store all user data:
    ```jsx
    const [user, setUser] = useState({
      name: "",
      email: "",
      password: ""
    });
    ```
  * Updated user data on input change:
    ```jsx
    const handleChange = (e) => {
      setUser({ ...user, [e.target.name]: e.target.value });
    };
    ```
  * Understood how to send this data to backend on form submit
