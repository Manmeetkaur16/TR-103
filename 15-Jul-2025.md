  * Understood that JSX is a syntax extension that allows writing HTML-like code inside JavaScript  
  * Learned JSX rules:

    * `className` instead of `class`  
    * JavaScript expressions inside `{ }`  
  * Practiced writing simple JSX elements and embedding variables  

* **Virtual DOM**

  * Studied how Virtual DOM improves performance  
  * Flow learned:
    * React updates Virtual DOM first  
    * Compares differences with actual DOM  
    * Updates only the changed parts (efficient rendering)  
  * Understood why React apps feel fast even with many UI updates
